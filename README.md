AutoReport AI

AutoReport AI is a simple SaaS MVP that generates automated business reports from CSV files.

Features
- Upload CSV files
- Automatic data analysis
- Chart generation
- PDF report generation

Tech Stack

- Python
- FastAPI
- Pandas
- Matplotlib
- ReportLab

  Go to project folder

- cd AutoReportAI
- Install dependencies
- pip install -r requirements.txt
- Run server
- uvicorn app.main:app --reload
- Open in browser
- tp://127.0.0.1:8000

This project can become a real micro-SaaS product if you add:

AI insights
automated dashboards
scheduled reports
